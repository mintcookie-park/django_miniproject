# django_miniproject
django를 활용한 블로그 개발 프로젝트입니다.

# 프로젝트 소개
PC 게임인 로스크아크의 직업간 밸런스, 팁, 정보를 공유하는 것을 목표로 만든 블로그 형식의 프로젝트로
회원인증을 통해 로그인을 한 후 글 및 댓글을 작성하여 서로 아이디어를 공유할 수 있도록 설계했습니다.

# 디렉터리 구조
디렉터리 파일 구조로는 아래와 같이 진행하였습니다.

mysite/

├── accounts

├── blog

├── main

├── media

├── static

├── templates

├── tutorialdjango
